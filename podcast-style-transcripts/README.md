Drop style reference transcripts into this folder.

Supported files:
- `.txt`
- `.md`

Recommended format:
```text
Speaker Name: Line of dialogue
Speaker Name: Another line
```

If the source transcript is unlabeled by speaker, the style profile will still use it for:
- overall energy
- cadence
- banter density
- segment shape

But labeled transcripts work better for role-specific host dynamics.
